Detailed Description

Parser class for integers.

IntArg(base) reads integers in base base. base defaults to 0, which means arguments are parsed in base 10 by default, but prefixes 0x, 0, and 0b parse hexadecimal, octal, and binary numbers, respectively.

Integer overflow is treated as an error.
